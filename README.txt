This is a demo of Hamage, https://github.com/slinkp/hamage

To run it:
Virtualenv recommended.
Minimum python version unsure, but probably 2.6.

First install hamage. Then python setup.py develop in here.

Then you can run django under wsgi like:

  python wsgi.py

Edit the wsgi file to play with the hamage configuration.

There is also a slide deck about the project in openoffice/libreoffice
format, slides.odp.
